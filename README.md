# ADXtools

Command-line tools to work with Criware ADX

## Build and Run
    make

## Usage
    Usage: adxtools [COMMAND] [OPTIONS] input
    Commands:
	    - adx2wav
		    Convert ADX to WAV

	    - wav2adx
		    Convert WAV to ADX

## Troubleshooting
  - If ADX output is noisy, try raising the highpass frequency
